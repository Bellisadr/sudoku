# Jogo de Sudoku - Desafio DIO

Este projeto é uma implementação de um Jogo de Sudoku em Java, desenvolvido como parte de um desafio de projeto da Digital Innovation One (DIO).

O projeto foi "forkado" do repositório original da DIO (https://github.com/digitalinnovationone/sudoku), adicionei os argumentos proposto e criei o botão dica como um sistema de dicas para o jogador.

## Funcionalidade Destaque: Botão de Dica

Além dos requisitos originais, este projeto implementa um botão "Dica".

Ao ser clicado, o sistema busca a primeira célula que está vazia ou preenchida incorretamente.

A célula é preenchida com o valor correto.

Para diferenciar visualmente, o número da dica é exibido em cor azul e o campo é desabilitado, tratando-o como um novo número "fixo".
